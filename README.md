Protohackers Perl solution system
----------

Development dependencies:
- Rex
- Rex::Commands::PerlSync
- Yath
- Code::TidyAll
- Perl::Tidy
- Perl::Critic

## Rex deployment

command: `rex -u USER -H HOST deploy`

## Testing

command: `./test`

## Running

command: `script/server --port=PORT --module=NAME_OR_NUMBER`
